experimental

Table of Contents

    2. Lists of Block Tags

Block Tags are used for various block properties. They can be called when testing for block arguments in commands. Some tags are prefixed with can_be, or can_hurt. One would expect these to have a boolean response.

Application #

Block tags can be applied exactly as items, as seen below:

Copy
{
	"format_version": "1.16.100",
	"minecraft:block": {
		"description": {
			"identifier": "wiki:tree_stump"
		},
		"components": {
			"tag:wood": {},
			"tag:wiki:very_useless": {}
		},
		"events": {}
	}
}
