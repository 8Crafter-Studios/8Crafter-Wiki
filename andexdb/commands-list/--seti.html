\\seti

Sets the blocks between the selected area to the selected block type with the specified integrity.

\\seti
Dflt USM Sec Lvl: WorldEdit
Dflt Required Tags: canUseChatCommands
Categories:system, world, server, worldedit
Version:1.0.0
Undo Supported:Yes
Functional:Yes
Deprecated:No

Syntax ​

\\seti [-c] <integrity: number> <blockPattern: BlockPattern>
