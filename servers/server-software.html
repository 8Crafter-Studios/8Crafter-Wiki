Minecraft servers allow players to play online or via a local area network with other people. This is very common within Java Edition Minecraft, but is also possible on Bedrock. You can download the official Mojang BDS software here..

Alongside the Vanilla BDS offering, many community projects exist, in a variaty of languages.
